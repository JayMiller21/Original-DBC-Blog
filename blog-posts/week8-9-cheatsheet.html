String

A string is a chain of characters that usually forms human readable words surrounded by either single or double quotes. A string can be empty, which would just be a pair of double/single quotes and no character inside. String manipulation is an important part of computer science, and there are a number of methods in Ruby that can be used on String class. These include but are not limited to the following.

String#print

Outputs the string to the console

puts "Howdy partner!" => "Howdy partner!"

String#puts

Outputs the string to the console followed by a line break.

puts "Howdy partner!" => "Howdy partner!"
#new line

String#concat

string1.concat(string2) appends string2 to the end of string1

"good ".concat("times") => "good times"

Alternatively, the '+' or '<<' also will perform a similar function and may even give you a slight performance boost!

"good " + "times" => "good times"
"good " << "times" => "good times"

String#ord

When performed on a single character, returns that character's ASCII value. When performed on a multi-character string, returns the ASCII value of the first letter in the string.

"a".ord => 97
"apple".ord => 97

String#to_sym

Returns a symbol object that cooresponds to the original string.

"coolbeans".to_sym => :coolbeans

For a string with spaces, the symbol name will be in quotes.

"cool beans".to_sym => :"cool beans"

Symbols are very similar to strings, execept that they cannot be changed once they are created.

foo = "fighters".to_sym
foo + " rool!" => Throws an Undefined Method Error

Integer

Integers are values that are represented rounded down to the nearest whole number. The following are some of Integer's most common methods.

Integer#to_s

Turns an integer into a string containing the digits as a corresponding series of characters.

963.to_s => "963"

Integer#chr

Returns the cooresponding character for the ASCII value.

97.chr => "a"

Mathematical Operations (+, -, *, /)

Preform the specified mathemaitical function. Return integers for operations involving only integers. Return floats for operations involving one or more floats.

1 + 2 => 3
1.0 + 2 => 3.0
5 - 2 => 3
5.0 - 2 => 3.0
1 * 2 => 3
1.0 * 2 => 3.0
5/2 => 2
5.0/2 => 2.5

Modulus (%)

The modulus operator returns the remainder of dividing one integer by another.

5 % 2 => 1
